# formula_site

基于 **Python API + CSV 文件数据库 + 动态页面** 的配方站。

## 功能
- 根路径：`/12sagittarius_ghpishbc/`。
- 首页：顶部搜索栏 + 右侧“添加”按钮。
- 搜索安全：查询词统一转义处理，避免特殊字符影响页面与匹配过程。
- 搜索能力：支持相似搜索（拼写接近也可命中）。
- 首页列表：展示所有配方名称与简要内容，点击整行进入详情。
- 详情页：
  - 左侧可编辑配方名称和配方内容。
  - 左侧底部包含“保存”和“删除”。
  - 右上角有“返回”按钮。
  - 右侧评论区支持前端即时提交（不刷新页面，不保存配方）。
- 点击“保存”后持久化到 CSV 并自动返回首页。

## 启动

python app.py
